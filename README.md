# The Studio Portfolio

A modern, responsive portfolio website built with React and Vite to showcase creative projects and studio work.

## Features

- **Project Showcase** - Display projects with images, descriptions, and details
- **Search Functionality** - Filter and search through projects
- **Add/Edit Projects** - Form interface to add new projects or edit existing ones
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Fast Performance** - Built with Vite for lightning-fast development and production builds

## 🛠️ Tech Stack

- **React 19** - UI library
- **Vite** - Build tool and development server
- **CSS3** - Styling (with modular CSS)
- **ESLint** - Code linting and formatting
